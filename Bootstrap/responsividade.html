Responsividade

Adapte seu site para todos os tamanhos de tela com facilidade

O que é responsividade?

Responsividade é a capacidade de um site se adaptar a diferentes tamanhos de tela, como celulares, tablets, notebooks e monitores grandes. O Bootstrap utiliza um sistema de grid flexível com 12 colunas que facilita a criação de layouts adaptáveis.

Breakpoints do Bootstrap

Breakpoints são os tamanhos de tela em que o layout "quebra" para se adaptar. Veja os principais:

Classe Tamanho mínimo Descrição
col- 0px Extra pequeno (mobile)
col-sm- 576px Pequeno (celulares grandes)
col-md- 768px Médio (tablets)
col-lg- 992px Grande (notebooks)
col-xl- 1200px Extra grande (monitores full HD)
col-xxl- 1400px Ultra grande (telas 2K+)
