O que é o Bootstrap?

Aprenda como criar sites responsivos e modernos com este poderoso framework.

Introdução

Bootstrap é um framework front-end gratuito e de código aberto usado para desenvolver sites responsivos e mobile-first com HTML, CSS e JavaScript.
Ele fornece uma coleção de estilos prontos, componentes e funcionalidades interativas que aceleram o desenvolvimento web.

Como usar

Você pode usar o Bootstrap de duas formas:

  • Via CDN (como estamos fazendo neste projeto)
  • Fazendo o download e hospedando os arquivos localmente

Veja como fazer a configuração via CDN:

1️⃣ Insira a seguinte linha de código no <head> logo após o <title> (CSS do Bootstrap): 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
2️⃣ Insira a seguinte linha de código antes de fechar o </body> (JS do Bootstrap): 

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
